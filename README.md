# HR-Analytics-Dashboard
End-to-end HR Analytics project using Power BI, featuring workforce analysis, salary insights, employee performance metrics, and interactive dashboards.

## Project Overview
- This project analyzes workforce distribution, employee performance, leave patterns, and salary trends across multiple departments and countries.

## Business Problem
- The HR department lacked a centralized view of workforce metrics, making it difficult to monitor employee trends and support strategic decision-making.


## Tools Used
- Power BI
- Power Query
- DAX

## Key Insights
- Manufacturing represents the largest workforce segment.
- Egypt has the highest employee count.
- Salary distribution varies significantly across departments.
- Employee growth peaked in 2019.
- No significant relationship was found between overtime hours and sick leave rates.
